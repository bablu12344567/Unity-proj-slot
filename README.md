# 🎰 Unity Slot Game Assignment

## 🎮 Overview
This is a Unity-based Slot Machine game developed as part of an assignment.  
The game uses RNG-based logic, smooth reel animations, and simple win conditions.

## 🧩 Features
- Slot machine with multiple reels
- Random Number Generator (RNG) for outcomes
- Win condition: all symbols match
- Smooth spinning animations
- Clean UI layout and alignment

## 🧠 Approach
Each reel generates a random value independently.  
After all reels stop, values are compared.  
If all match → player wins.
